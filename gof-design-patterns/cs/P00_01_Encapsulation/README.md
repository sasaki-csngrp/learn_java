C言語版の`BankAccount.c`を作成しました。

この実装では、構造体と関数ポインタでJavaのクラスに相当するカプセル化を実現しています：

1. **構造体によるデータカプセル化**: `BankAccount`構造体に`balance`を保持
2. **関数ポインタによるメソッド**: `deposit`、`withdraw`、`getBalance`、`destroy`を関数ポインタとして定義
3. **コンストラクタ相当**: `BankAccount_new()`でオブジェクトを生成し、関数ポインタを初期化
4. **メソッド呼び出し**: `account->deposit(account, 500)`のように関数ポインタ経由で呼び出し

コンパイルと実行の例：
```bash
gcc -o BankAccount BankAccount.c
./BankAccount
```

実行すると「残高: 1300.00」と表示されます（初期残高1000 + 預金500 - 引き出し200）。

より厳密なカプセル化が必要な場合は、ヘッダーファイル（.h）と実装ファイル（.c）を分離し、不完全型を使って`balance`への直接アクセスを防ぐ方法もあります。
