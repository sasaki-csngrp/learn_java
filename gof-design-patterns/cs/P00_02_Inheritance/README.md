C言語版の`Dog.c`を作成しました。C言語でも継承相当を実装できます。

この実装では、以下の方法で継承を実現しています：

1. **構造体の埋め込み**: `Dog`構造体の最初のメンバーとして`Animal`構造体を含めることで、継承を実現
   - これにより、`Dog*`を`Animal*`に安全にキャスト可能

2. **関数ポインタによる仮想関数**: `makeSound`を関数ポインタとして定義し、オーバーライドを実現
   - 基底クラス（`Animal`）では`animal_makeSound`を設定
   - 派生クラス（`Dog`）では`dog_makeSound`を設定

3. **ポリモーフィズム**: 基底クラスのポインタ経由でも、正しい派生クラスのメソッドが呼ばれる

コンパイルと実行の例：
```bash
gcc -o Dog Dog.c
./Dog
```

実行すると：
```
ポチがワンワンと鳴く
ポチがワンワンと鳴く
```

**C言語での継承の特徴**：
- 構造体の埋め込みにより、メモリレイアウトが保証される
- 関数ポインタにより、実行時の多態性を実現
- 型安全性はコンパイラでは保証されないため、実装に注意が必要

C言語でも継承は実現可能ですが、Javaのような言語サポートはなく、手動で実装する必要があります。この実装は、オブジェクト指向の継承の概念をC言語で表現したものです。
